# Java Practice Examples

A collection of Java practice programs demonstrating various fundamental concepts.

## Examples Included

- **datatypesex.java** - Demonstrates Java data types and their usage
- **arrayex1d.java** - Shows 1D array operations and manipulation
- **typecastingex.java** - Illustrates type casting concepts in Java
- **example.java** - Basic Java programming example
- **example2.java** - Additional Java programming example

## Prerequisites

- Java Development Kit (JDK) installed
- Java compiler (`javac`) available in PATH

## How to Compile and Run

### Compile all Java files:
```bash
javac *.java
```

### Run specific examples:
```bash
# Data types example
java datatypesex

# 1D Array example
java arrayex1d

# Type casting example
java typecastingex

# Basic examples
java example
java example2
```

## Learning Objectives

These examples cover:
- Basic Java syntax
- Data types and variables
- Array operations
- Type casting
- Object-oriented programming basics
